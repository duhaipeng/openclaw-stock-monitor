# 股市监控技能

实时监控股票价格，设置价格提醒，提供技术分析。

## 功能特点

### 📊 实时监控
- 支持A股、港股、美股
- 实时价格查询
- 涨跌幅计算
- 成交量分析

### ⏰ 智能提醒
- 价格突破提醒
- 涨跌幅提醒
- 定时行情推送
- 技术指标提醒

### 📈 技术分析
- 简单移动平均线
- 相对强弱指数(RSI)
- 支撑阻力位识别
- 趋势判断

## 快速开始

### 安装依赖
```bash
pip install pandas numpy
```

### 基本使用

**查询单只股票：**
```
查询贵州茅台股价
```

**设置价格提醒：**
```
当贵州茅台突破2000元时提醒我
```

**查看技术指标：**
```
分析腾讯控股的技术指标
```

## 配置说明

在TOOLS.md中添加：
```markdown
### 股票监控
- 默认监控列表: ["000001", "399001", "HSI", ".DJI"]
- 提醒阈值: 涨跌5%提醒
- 检查频率: 每30分钟
```

## 高级功能

### 自定义监控列表
支持JSON格式配置文件：
```json
{
  "stocks": ["000001", "00700", "AAPL"],
  "alert_rules": {
    "price_break": true,
    "change_percent": 5,
    "volume_surge": true
  }
}
```

### 技术指标计算
- MA5, MA10, MA20 移动平均线
- RSI(14) 相对强弱指数
- 布林带分析
